# 常用 Linktree 类聚合页服务

查询人物社交账号分布时，尝试以下 URL 格式：

| 服务 | URL 格式 | 特点 |
|------|----------|------|
| **Linktree** | `linktr.ee/{username}` | 最主流，覆盖面最广 |
| **bio.link** | `bio.link/{username}` | 简洁，欧美常用 |
| **Beacons** | `beacons.ai/{username}` | 创作者常用，支持收款 |
| **Carrd** | `{username}.carrd.co` | 轻量个人页，韩国/日本常见 |
| **lit.link** | `lit.link/{username}` | 日本主流 |
| **Fanlink** | `fanlink.tv/{username}` | 面向 DJ/音乐人 |
| **Linklist** | `linklist.bio/{username}` | 亚洲常见 |
| **Allmylinks** | `allmylinks.com/{username}` | 欧美常见 |
| **linkbio.co** | `linkbio.co/{username}` | 常见聚合服务（如 babbyang 实际使用） |
