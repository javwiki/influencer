# 유리사

## 简介

韩国模特、Cosplayer、歌手、YouTuber。1992年出生于首尔。以Lolita fashion和Cosplay闻名，曾活跃于中韩两国市场。Mensa会员（IQ 156）。出演Netflix生存综艺《Bloody Game 2/3》及《Death Game 2》并夺冠。
