# 长泽茉里奈

## 简介

日本グラビアアイドル（写真偶像）、演员、艺人、プロ雀士（专业麻将选手）。1995年出生于埼玉县熊谷市。以童颜F杯罩的"合法ロリ巨乳"著称，爱称「まりちゅう」。元放課後プリンセス成员。妹妹为长泽圣爱。现所属TRUSTAR。同时以自身名义制作偶像团体Chuu♡Cute。
